# 🍹 Smoothify

**Smoothify** is a responsive smoothie-themed landing page built with **HTML, CSS, and JavaScript**.  
It includes interactive features like:  
- 🥤 Product cards with animations  
- 💬 AI-style chatbox  
- 🔐 Login & Signup forms  
- ❓ FAQ accordion  
- 📱 Fully mobile responsive design
